# Perform Windows data backups
<h1> creating copies of your files, settings, and system information</h1>

 ### [YouTube Demonstration] in progress!!
 
<h2>Description</h2>
Creating copies of your files, settings, and system information to protect against data loss from corruption, ransomware, or hardware failure, the Windows Backup app for settings/apps, and File History or Backup and Restore (Windows 7) for local external drive backups.

<h2> Skills Learned</h2>

- <b> Backup frequency and retention period </b>
- <b> Be able only to include essential folders to be  backed up </b>
- <b> Be able to backup files temporarily and exclude non business critical files </b>

<h2>Languages and Utilities Used</h2>

- <b> Windows Server simulator</b>

- <b> Able to check if logs are free from errors and any potential issues </b>
