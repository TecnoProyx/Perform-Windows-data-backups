# Perform Windows data backups
<h1> creating copies of your files, settings, and system information</h1>

 ### [YouTube Demonstration] in progress!!
 
<h2>Description</h2>
Creating copies of your files, settings, and system information to protect against data loss from corruption, ransomware, or hardware failure, OneDrive for cloud syncing, the Windows Backup app for settings/apps, and File History or Backup and Restore (Windows 7) for local external drive backups.
